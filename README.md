# Twich-Alert

Windows-приложение для запуска Telegram-бота оповещения о стримах.

## Возможности

- 🔔 **Для стримеров**: привязка Twitch-канала, настройка оповещений, отправка "Стрим начался!" подписчикам
- 👀 **Для зрителей**: подписка на любимых стримеров, включение/отключение оповещений
- 📊 **Счётчик переходов**: отслеживание количества переходов по ссылкам в оповещениях
- 💡 **Обратная связь**: возможность предложить новую функцию разработчику
- ⚙️ **Админ-панель**: просмотр статистики и рассылка сообщений (TG ID: 450146311)

## Установка

1. Скачайте `Twich-Alert.exe` из папки `dist`
2. Запустите приложение
3. Введите токен Telegram-бота (полученный от @BotFather)
4. Нажмите "Запустить"

## Использование

### Для стримеров:
1. Нажмите /start → "Я стример"
2. Пришлите ссылку на ваш Twitch-канал
3. Настройте текст оповещения
4. Когда начнёте стрим → нажмите "Стрим начался!" → подтвердите отправку

### Для зрителей:
1. Нажмите /start → "Я зритель"
2. Пришлите ссылку на канал стримера
3. Включите/выключите оповещения по желанию

## Требования

- Windows 10/11
- Python не требуется (приложение уже скомпилировано)

## Сборка из исходников

```bash
pip install -r requirements.txt
pyinstaller --onefile -w --exclude-module setuptools --exclude-module pkg_resources main.py -i icon.ico -n Twich-Alert
```
